\section{Choix Techniques}

\subsection{Visual Studio Community}

Microsoft Visual Studio \cite{6} est une série de programmes développé par Microsoft.

\medskip



C'est une suite complète qui génère des applications grâce à ces divers outils de développement. Visual C\#, Visual Basic, Visual C++ exploitent tous l'IDE (Integrated Developement Environment) qui nous permettrons de distribuer les outils et simplifie la réalisation des solutions en utilisant les divers langages. 

\medskip

\begin{center}
\includegraphics[scale=0.15]{Images/logovisualstudio}
\end{center}


\medskip

Pour ce projet, nous avons utilisé la version Visual Studio Community 2019 avec la langage Visual Basic \cite{7} \cite{8} qui est connu pour ça structure simple et pratique pour le code exécutable.




\subsection{Android Studio}

Android Studio \cite{9} est un logiciel officiel conçu par Google en 2013 pour le développement des applications Android, basé sur l'IDE (Integrated Developement Environment) IntelliJ IDEA et disponible gratuitement à télécharger sous les systèmes d'exploitation Windows, macOS, Chrome OS et Linux.

\medskip



Ce logiciel permet en majorité d'éditer les fichier Java, Kotlin et XML d'une application Android.

\medskip


En plus du puissant code éditeur et des outils des développement, Android studio offre plus de fonctionnalités comme flexibilité de construction basé sur Gradle, un émulateur rapide et riche de fonctionnalités, des outils de test étendus, tous avantages qui améliorent la productivité quand nous construisons un projet Android apps.

\medskip


Pour notre projet, nous avons utilisé la version 4.1.3 \cite{10} avec le langage Java \cite{11}.





\subsection{MySQL Workbench}

MySQL Workbench \cite{12} \cite{13} est un logiciel de gestion et d'administration de bases de données MySQL crée en 2004, disponible sur Windows, Linux et MAC OS X.

\medskip



Ce logiciel nous permettra de la création, modification ou de la suppression des tables, des comptes et d'effectuer toutes les opérations de gestion d'une base de données grâce à son interface graphique qui est clair et facile à utilisé. 



\subsection{Firebase Platform}

Firabase \cite{14} est un site qui offre plusieurs divers services d'hébergement lancée en 2011 sous le nom "Envolve" par Andrew Lee et James Templin, puis le service est racheté par Google en octobre 2014 et mis désormais à la maison mère de Google : Alphabet.


\medskip


Toutes les implémentations et la gestion des serveurs de Firebase sont de la responsabilité exclusive d'Alphabet. Les applications utilisant Firebase intègrent une bibliothèque qui permet diverses interactions possibles.


\medskip

\begin{center}
\vspace*{-0.5cm}
\includegraphics[scale=0.2]{Images/logofirebase}
\vspace*{-0.5cm}
\end{center}


\medskip


Ce site aujourd'hui offre la possibilité d'héberger toute sorte type d'application (Android, iOS, JS, Node.js, Java, Unity, PHP, C++ etc..) aussi que d'autres fonctionnalités comme l'authentification sociale ou la sauvegarde des contenus dans des bases de données NoSQL ou temps réel \cite{15}.



\medskip



\medskip



Pour notre projet nous utilisons la fonctionnalité d'hébergement dans une base de données en temps réel.

\medskip



\subsection{Photofiltre}

Photofiltre est un logiciel d'édition d'image gratuit créé par Antonio Da Cruz. En plus du traitement d'image classique (recadrage, contraste, gamma, etc.), il dispose également de filtres pour améliorer et convertir les photos numériques.

\medskip


Photofiltre \cite{16} est la version gratuite du partagiciel Photofiltre Studio et nous avons choisi ce logiciel pour créer quelque parties du design comme l'icône d'application.
